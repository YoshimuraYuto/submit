# README

概要
人生について考えるアプリ
私は自殺を進めている訳ではない。むしろこの世の中から無くなって欲しいと思う。だがなぜ、日本には自殺する人が多いのか。みんなに考えてもらうためにこのアプリを立ち上げた。
また、お金、家族、友達、哲学の観点から人生を謳歌してもらうために記事を書く所存である。
